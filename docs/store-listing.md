## 詳細說明（直接複製）

```
找工作時，看到一家有興趣的公司，你是不是還要另外開分頁去 PTT 翻半天，才知道這家到底風評如何？

「求職論壇風評」幫你把這個步驟省掉。

在 104 打開任何職缺或公司頁面，它會自動認出這家公司，並在頁面右側列出 PTT 上關於這家公司的討論——標題直接看，點一下就跳到原文。

■ 功能
・自動偵測 104 職缺／公司頁的公司名稱
・搜尋 PTT 科技業板、軟體工作板、薪資板、求職板
・結果依看板分組，標題可直接點開原文
・自動把「[新聞]」類文章收進獨立區塊（上市公司的股價／訴訟新聞往往和求職無關），需要時可展開
・一鍵跳轉 Dcard、Google 搜尋這家公司

■ 聰明的公司名比對
104 上是「台灣積體電路製造股份有限公司」，但 PTT 上大家叫「台積電」。本擴充內建對照與正規化規則，會自動用大家實際的叫法去搜尋，才找得到真正的討論。

■ 隱私
・不收集、不傳送、不販售任何個人資料
・沒有帳號、沒有後端伺服器、沒有追蹤統計
・所有處理都在你的瀏覽器本機完成
・唯一的對外連線，是讀取 PTT 的公開搜尋頁面

■ 說明
・本擴充只顯示文章標題與連結，內容版權屬於原作者，點擊後導向 PTT 原文閱讀。
・討論內容為網友個人意見，僅供參考。
```

## 單一用途說明（Single purpose，審查必填）

```
本擴充的單一用途是：在台灣 104 人力銀行的職缺與公司頁面上，顯示該公司在 PTT 論壇上的相關討論連結，協助求職者查詢公司風評。
```

## 權限理由（審查必填，這裡最常被打回）

**`host_permissions: *://*.ptt.cc/*`**
```
用於向 PTT（ptt.cc）的公開搜尋頁面讀取文章列表，以取得使用者目前查看之公司的相關討論標題與連結。僅讀取公開內容，不傳送任何使用者資料至 PTT 或任何第三方。
```

**content script 於 `*://www.104.com.tw/job/*`、`*://www.104.com.tw/company/*`**
```
用於在 104 職缺／公司頁面讀取該頁的公司名稱（取自頁面公開的結構化資料），並在頁面上插入顯示搜尋結果的面板。僅在這兩類頁面執行，不讀取或傳送使用者的個人資料、履歷或帳號資訊。
```
